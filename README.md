# Portfolio Website

## Overview

This repository contains my personal portfolio website, hosted using GitHub Pages.

The website presents my work as a Game Development student, including my projects, skills, and links to my professional profiles.

## Purpose

The goal of this portfolio is to:

- Present my development projects in a structured way
- Showcase my skills in game development and programming
- Provide a central hub for my GitHub, LinkedIn, and other platforms
- Demonstrate web development abilities

## Content

The website includes:

- Personal introduction
- Project showcase
- Links to GitHub and LinkedIn
- Overview of my skills and interests
- Access to selected development projects

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages for hosting

## Hosting

This website is deployed using GitHub Pages.

## Projects Featured

- Eagle Vision (Minecraft mod)
- Javascript Tools (collection of scripts and utilities)
- Additional development experiments

## Design Goals

The design focuses on:

- Simplicity and readability
- Fast access to projects
- Clean presentation of developer profile
- Easy navigation
